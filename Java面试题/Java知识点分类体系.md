## 💡 学习建议

### 初学者路线
1. **第一阶段**：卷I 第1-5章（语言基础 + 面向对象）
2. **第二阶段**：卷I 第7-9章（异常 + 泛型 + 集合）
3. **第三阶段**：卷I 第14章 + 卷II 第1-2章（并发 + Stream + IO）
4. **第四阶段**：卷II 第4-6章（网络 + 数据库 + 日期时间）

### 面试准备路线
1. **核心基础**：数据类型、面向对象、集合、异常
2. **并发编程**：线程、锁、线程池、并发工具类
3. **JVM 相关**：类加载、内存模型、GC（需额外学习）
4. **新特性**：Lambda、Stream、Optional、新日期 API
5. **框架基础**：反射、注解、动态代理

### 实战项目路线
1. **基础项目**：控制台应用（集合 + IO）
2. **进阶项目**：多线程下载器（并发 + 网络）
3. **高级项目**：Web 应用（JDBC + Servlet + 框架）
4. **企业项目**：分布式系统（Spring 全家桶）
